Вопрос - Приведите пример использования Nullable типа при проектировании класса.
Класс не используется для работы с записями БД и не используется для их представления. Обоснуйте свой выбор.

Ответ - Nullable типы можно использовать для структур, состояние которох может быть не инициализировано при создании экземпляра класса.
Т.е. например, поле int? Age в профиле пользователя может быть и не заполнено и его состояние лучше представить как null
